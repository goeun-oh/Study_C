## 🛒 **E-Commerce Platform "ShopEasy"**

### 📋 프로젝트 개요
완전한 기능을 갖춘 온라인 쇼핑몰 플랫폼으로, 사용자와 관리자 모두를 위한 직관적인 인터페이스를 제공합니다.

### 🛠️ 기술 스택
- **Frontend**: React 18, TypeScript, Styled-components
- **Backend**: Node.js, Express.js, JWT Authentication
- **Database**: MongoDB, Redis (세션 관리)
- **Payment**: Stripe API, PayPal SDK
- **Deployment**: AWS EC2, S3, CloudFront

### ✨ 주요 기능
- **사용자 기능**: 회원가입/로그인, 상품 검색/필터링, 장바구니, 주문 내역
- **결제 시스템**: 다중 결제 수단 지원, 실시간 결제 상태 업데이트
- **관리자 대시보드**: 상품 관리, 주문 관리, 매출 통계, 사용자 관리
- **반응형 디자인**: 모바일, 태블릿, 데스크탑 완벽 지원

### 📊 성과
- **성능**: 페이지 로딩 속도 2초 이내 달성
- **사용성**: 구매 전환율 15% 달성

- **Database**: PostgreSQL, Redis

- **실시간 메시징**: 1:1 채팅, 그룹 채팅 (최대 100명)

- **사용자 상태**: 온라인/오프라인 상태, 마지막 접속 시간
- **알림 시스템**: 브라우저 알림, 이메일 알림
- **메시지 암호화**: End-to-end 암호화 지원
